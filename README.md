# 微信小程序仿网易云基本介绍

该项目使用小程序全新开发模式：前端组件化开发+后端云开发全新开发模式，分为小程序端与CMS端

***

其中小程序端有许多自定义组件
- 音乐功能
    - 歌单组件

    - 歌曲列表组件

    - 歌曲进度条组件

    - 歌词组件

- 博客功能
    - 博客卡片组件

    - 博客控制组件

    - 登录授权组件

    - 模糊搜索组件
